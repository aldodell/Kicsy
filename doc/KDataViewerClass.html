Class: KDataViewerClass

KDataViewerClass(…args)

Class for KDataViewerComponent. This class represents a container component that can contain other components, but only one. The child components are stored in the root component and could have children. Is very important that children nodes will be near from this component.

        let viewer;
        let arrayData2;

        //Note KDataViewer wrapping only ONE KLayer wich has two KText
