Methods

broadcast() → {void}

Broadcasts the message to all the applications registered in Kicsy.applications. This function iterates over the applications in the Kicsy.applications array and calls the processMessage method of the application with the current instance of the KMessageClass as an argument. The processMessage method processes the message and returns the result. If the target application is found, the processMessage method is called and the result is returned.
