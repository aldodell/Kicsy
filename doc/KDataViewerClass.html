Adds the provided component to the data viewer. This function is a convenience wrapper around the setRootComponent function. It adds the provided component to the data viewer by calling setRootComponent with the provided component. This allows the data viewer to be created and have the root component set all in one go.

setRootComponent(component) → {KDataViewerClass}

Sets the root component of the data viewer. This function sets the root component of the data viewer to the provided component. The root component is a KicsyVisualContainerComponent instance that is used to house all of the components that are to be viewed in the data viewer. Only one root component can be set at any given time.
